# Recette

Small assignment III - Recette

### How to run

1. open 2 terminals

| terminal 1      | terminal 2         |
| --------------- | ------------------ |
| run the api     | run the web server |
| -----------     | ------------------ |
| `cd recipe-api` | `cd my-app`        |
| `npm run dev`   | `npm start`        |

2. open `http://localhost:3000` in your browser
